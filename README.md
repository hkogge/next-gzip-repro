## Getting Started

First, run the development server:

```bash
npm run dev
```

- Open old API route at [/api/foobar](http://localhost:3000/api/foobar): ✅ gzip compressed response
- Open new route handler at [/foobar](http://localhost:3000/foobar): ❌ no compression
